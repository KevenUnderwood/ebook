### Key point

1.  a project built by vue
2.  rem-based layout
3.  Scss
4.  Responsive Design
5.  webpack
6.  epub.js

### Function 
1.  previous page/ next page
2.  show menu & setting 
    1.  Index
    2.  jump to by progress bar
    3.  Theme
    4.  Font size
---
### Usage
#### install dependencies
`npm install`

#### serve with hot reload at localhost:8080
`npm run dev`

#### build for production with minification
`npm run build`
